Quem somos? A Equitech é um projeto de dois estudantes do CEFET-MG Divinópolis que possui o objetivo de transformar o acesso a conteúdos sobre programação mais acessíveis para todos os interessados em aprender. Por que escolher a Equitech?  Acesso completamente gratuito Todo o conteúdo disponibilizado pela Equitech é gratuito, sendo pago apenas caso o usuário opte por apoiar o projeto.  Conteúdo completo e de qualidade Como programadores, entendemos que, para iniciantes, o assunto pode ser complicado. Por isso, priorizamos qualidade, com aulas objetivas e bem explicadas.  Projetos de expansão No aprendizando a programação, a prática é um fator primordial para a expansão do conhecimento. Tendo isso em vista, disponibilizamos dezenas de projetos gratuitos com correção.
Cursos disponíveis
    • HTML
    A Hypertext Markup Language é a linguagem de marcação que estrutura todas as páginas web, essencial para qualquer desenvolvedor que deseja trabalhar no ramo de web development. No curso de HTML, você irá aprender os fundamentos da programação web, bem como estruturas básicas como parágrafos, títulos, listas, tables, divs, frames e muito mais.
    É o curso básico da Equitech, sendo recomendado como ponto de partida para qualquer desenvolvedor.
    CSS
    Conhecido também como Cascading Style Sheets, o CSS é a linguagem utilizada para estilizar os websites, deixando-os mais agradáveis e chamativos. É essencial para que sua página web atraia novos usuários, dando um toque especial e deixando-o com cara de moderno. Aprenda sobre margin, padding, fontes, cores, animações e muito mais nesse curso da Equitech.
    Combine-o com HTML e veja seus sites ficarem cada vez mais bonitos e profissionais.
    Javascript
    Javascript é a linguagem mais famosa e utilizada no mundo todo. Na Equitech, você irá aprender a utilizá-la para incrementar funções no seu website, deixando-o mais interativo. Estude sobre arrays, animations, forms, intevalos, variáveis, e sobre as mais diversas funções da própria linguagem.
    Combinado com CSS e HTML, o conhecimento sobre Javascript o tornará um desenvolvedor front-end profissional.
    C
    "Se você aprende C, aprende qualquer outra!" Essa frase dita por muitos programadores é verdade! A linguagem C é conhecida como o pai das linguagens, tendo sido utilizada como base para muitas outras, como C++, C#, Java, entre outras. É por isso que aprender C é essencial para qualquer programador.
    Estude estruturas, ponteiros, TAD's, recursividade, vetores, matrizes e muito mais no curso completo de C da Equitech.
    Python
    Sendo conhecido como a famosa linguagem da automação, Python é, sem dúvidas, uma das melhores portas de entrada para qualquer programador iniciante, por ser extremamente documentada e fácil de aprender.
    No curso de Python, você irá desde os fundamentos aos conceitos mais avançados da linguagem, se tornando um profissional da automação.

    Perguntas frequentes
    1. Como surgiu o projeto?
    A Equitech surgiu como um projeto de trabalho bimestral do Prof. Alberto Pena Lara do CEFET-MG Campus V.
    2. Quem somos?
    Nós, da Equitech, somos dois alunos do CEFET-MG Campus Divinópolis. Somos Thales Alves Amaral e Pedro Henrique Resende dos Santos, do 1° ano de Informática.
    3. Os cursos são gratuitos?
    Sim! Todos os cursos disponibilizados na Equitech são completamente gratuitos. Você só paga o plano premium para ter acesso à um site sem anúncios e monitorias exclusivas com professores da Equitech.
    4. Como funciona o premium?
    O plano premium funciona como uma assinatura. Ele é renovado mensalmente no valor de 19,99 até o cancelamento do usuário. Caso o plano for cancelado, o usuário continuará tendo os benefícios do premium até o fim do prazo do último pagamento.
